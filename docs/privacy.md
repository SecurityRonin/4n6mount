# Privacy Policy

*Last updated: 2026-07-24*

## Summary

4n6mount is a local Rust tool. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

4n6mount reads only the image and hash-database files you point it at, and writes only to the sidecar / session directories you specify. All processing happens on your local machine. Nothing is uploaded anywhere.

## Telemetry

4n6mount has **no telemetry**. It makes no network requests of any kind.

## Open Source

4n6mount is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/4n6mount](https://github.com/SecurityRonin/4n6mount).
